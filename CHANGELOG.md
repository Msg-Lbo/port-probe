# 更新日志

## v1.0.3 (2026-05-09)

- 更新按钮改为应用内下载：显示下载进度，下载完成后静默安装并重启程序。
- 更新检测增加超时处理，避免 GitHub 或更新日志地址不可达时一直停在“获取版本信息”。
- 支持在 `config.ini` 的 `[update]` 中配置 `manifest_url`，可指向内网 HTTP 更新清单，绕开 GitHub 和 Win7 TLS 环境问题。

## v1.0.2 (2026-05-09)

- 替换应用图标，安装包、桌面快捷方式和主程序窗口统一使用新图标。

## v1.0.1 (2026-05-09)

- 优化左下角版本号的更新状态提示，发现新版本时可直接看到目标版本号。
- 保持 Win7 RTM 发布包的 OpenSSL 运行库随包分发，确保检测更新可以正常访问 GitHub。

## v1.0.0 (2026-04-29)

- 仓库历史与标签已重置
